# A11y - An Audible Observibility Tool
An sound-based ambient monitoring tool for non-visual modality based monitoring of VM CPU load, RAM Usage and continuous error logging.  

This repository was created in order to develop, maintain and host documentation pertaining an academic research project undertaken by a student of the Disruptive DevOps module at The Atlantic Technological University.
