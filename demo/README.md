# Demo

`demo.mp4` — walkthrough of the Sonic A11y Grafana panel sonifying DevOps metrics.

## Note on audio quality

Any intermittent stutters or dropouts in the recording are an artefact of the capture environment, not the plugin itself:

- Recorded on an underpowered laptop running on battery (CPU throttled).
- OBS screen recording competes with the browser's Web Audio thread for CPU.
- Known Windows audio driver quirks under load on this machine.

The sonification renders cleanly in normal use; the artefacts are confined to this recording.
